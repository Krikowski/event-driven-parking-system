# Estapar Parking

Implementação do teste técnico backend para gerenciamento de estacionamento orientado a eventos.

## Estrutura da solução

- `src/Estapar.Parking.Api` — API HTTP
- `src/Estapar.Parking.Application` — casos de uso e contratos
- `src/Estapar.Parking.Domain` — entidades e regras de negócio
- `src/Estapar.Parking.Infrastructure` — persistência e integrações
- `tests/Estapar.Parking.UnitTests` — testes unitários
- `tests/Estapar.Parking.IntegrationTests` — testes de integração
